# ChatGPT_SignLanguage 

## 프로젝트 내용
>**수어로 ChatGPT를 사용하는 웹 서비스 데모 구현**   
>**개발기간 : 2023 - 05 - 17 ~ 2023 - 05 - 25**

<hr></hr>

## 프로젝트 소개
**1. ChatGPT**   
   ChatGPT API 연동하여 ChatGPT 기능 구현
   
**2. Signlanguage to Chatgpt**   
   ChatGPT API 연동하여 수어 사진 업로드 시 번역 모델 구현
